Utilizando o Jquery

O que é Jquery?

Jquery é uma 'biblioteca de JavaScript muito utilizada para fazer as mesmas coisas do JS, porém de uma fora mais prática e reduzida. Ela possui duas versões, a compactada, utilizada após o upload do site pelo seu tamanho reduzido, e a descompactada, utilizada durante o desenvolvimento do site por proporcionar a leitura feita pelo programador, porém é uma versão mais pesada.

Árvore DOM

A árvore DOM nada mais é do que a forma hierárquica com que seu site é representado, divisão de tags e a interpretação do código. Essa forma de ver o código é utilizada para entender e até mesmo manipular os elementos com o JavaScript/Jquery.

Seletores

Para manipular esses elementos DOM nós utilizamos o operador $(seletor), se quisessemos o id do elemento p: $(#paragrafo); ou acessar todos os elementos de uma classe chamada paragrafos: $(.paragrafos)

Exeplos: $("p") || $(".destaque"); || $("#menu"); || $("#menu a"); || $("#header div p span")
